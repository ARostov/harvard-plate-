

# Harvard Plate - Калькулятор гарвардской тарелки

Визуальный конструктор сбалансированного питания на основе принципов Гарвардской тарелки.

## Основные возможности

### Конструктор тарелки
- Визуальное представление Гарвардской тарелки (50% овощи, 25% белки, 25% углеводы)
- Drag & drop добавление продуктов
- Реальный расчет питательности
- Сохранение и загрузка тарелок

### Анализ питательности
- Детальный анализ БЖУ (белки, жиры, углеводы)
- Расчет калорийности
- Оценка по Гарвардской тарелке
- Графики и визуализация данных

### Анализ сочетаемости
- Проверка совместимости продуктов
- База знаний о хороших и плохих сочетаниях
- Рекомендации по улучшению
- Принципы сочетаемости вкусов

### База продуктов
- 100+ продуктов с полной информацией
- Поиск и фильтрация
- Добавление в избранное
- Быстрое добавление в тарелку

## Установка и запуск

### Требования
- Node.js 16+
- npm или yarn

### Установка
1. Клонирование репозитория
2. Установка зависимостей

Команды:
- npm install
- yarn install

### Запуск разработки
- npm run dev
- yarn dev

Приложение будет доступно по адресу: http://localhost:3000

### Сборка для production
- npm run build
- yarn build

Собранные файлы будут в папке dist/

## Структура проекта

Основные папки:
- public/ - Статические файлы
- src/ - Исходный код
    - components/ - Компоненты Vue
    - data/ - Локальная база данных
    - utils/ - Утилиты
    - styles/ - Стили
- package.json - Зависимости
- vite.config.js - Конфигурация Vite

## Технологический стек

- Vue 3 - основной фреймворк
- Vite - сборщик и dev-сервер
- Vue Router - маршрутизация
- LocalStorage - хранение данных пользователя
- CSS Custom Properties - система дизайна

## Принципы Гарвардской тарелки

Проект основан на принципах Гарвардской тарелки здорового питания:

1. 50% - овощи и фрукты
2. 25% - белки (мясо, рыба, бобовые)
3. 25% - углеводы (цельнозерновые)

Приложение помогает следовать этим принципам, анализируя состав тарелки.
